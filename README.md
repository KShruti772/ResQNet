# ResQNet Frontend

Modern React + Tailwind UI for ResQNet — Unified Smart Emergency Response System.

## Features
- Landing page with storytelling sections
- Login and register pages with role selection
- Responsive design for mobile and desktop
- Reusable components and premium SaaS-style UI
- Ready for backend integration

## Run locally

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Open the address shown by Vite in your browser.
